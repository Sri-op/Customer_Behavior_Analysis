# Customer Behavior Analysis Project

## Project Overview

This project analyzes customer transactions and purchasing behavior to identify customer segments, purchase trends, and churn risk patterns. The analysis helps generate business insights and recommendations to improve customer engagement and retention.

## Objective

* Perform data cleaning and preprocessing
* Engineer useful customer features
* Segment customers using RFM Analysis
* Analyze purchase behavior and retention trends
* Generate actionable business recommendations

---

## Dataset

Dataset used: Customer Behavior Analysis Dataset

Columns include:

* Customer ID
* Purchase Date
* Product Category
* Product Price
* Quantity
* Total Purchase Amount
* Payment Method
* Customer Age
* Returns
* Customer Name
* Gender
* Churn

## Project Workflow

### 1. Data Cleaning

* Removed duplicate records
* Handled missing values
* Converted date columns

### 2. Feature Engineering

* Created Total Spending feature
* Prepared customer-level metrics

### 3. Customer Segmentation

Performed RFM Analysis:

* Recency
* Frequency
* Monetary Value

Customer Groups:

* Champions
* Loyal Customers
* Potential Customers
* At Risk Customers

### 4. Visualization

Created:

* Customer Segment Distribution
* Purchase Trend Analysis
* Top Product Category Analysis
* Retention and Churn Insights

1. Reward loyal customers with exclusive offers.
